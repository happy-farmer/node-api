# node-api

> NodeJs based API service

Implementation of related part from [schemas](https://github.com/happy-farmer/schemas).

### Market

- POST  /v1/markets
- GET   /v1/markets

- GET   /v1/markets/:id
- PATCH /v1/markets/:id
- DELETE /v1/markets/:id

### Farm

- POST  /v1/farms
- GET   /v1/farms

- GET   /v1/farms/:id
- PATCH /v1/farms/:id
- DELETE /v1/farms/:id
